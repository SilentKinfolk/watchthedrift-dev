# watchthedrift-dev

Experimentation & preview repo for **watchthedrift** v2.

Production (stable, live) lives at
[SilentKinfolk/watchthedrift](https://github.com/SilentKinfolk/watchthedrift) →
<https://silentkinfolk.github.io/watchthedrift/>

This is where radical changes get tried and (once there's code) deployed to a
separate preview site, without touching production. Start with **[`SPEC.md`](SPEC.md)** —
the directional spec for v2, meant to be sharpened with `/grill-me`, turned into a
PRD with `/to-prd`, and built in a loop.

No app code yet — just the direction.
